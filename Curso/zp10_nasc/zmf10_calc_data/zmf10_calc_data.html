Description: Calculo de Data 
FUNCTION ZMF10_CALC_DATA.
*"----------------------------------------------------------------------
*"*"Interface local:
*"  IMPORTING
*"     REFERENCE(DATAI) TYPE  SY-DATUM
*"     REFERENCE(DATAF) TYPE  SY-DATUM
*"  EXPORTING
*"     REFERENCE(NDIAS) TYPE  I
*"  EXCEPTIONS
*"      EDATA1
*"      EDATA2
*"      EDATA3
*"----------------------------------------------------------------------

*       Global data declarations

" Datai vazia
IF datai is INITIAL.
    RAISE edata1.
ENDIF.

IF dataf is INITIAL.
  RAISE edata2.
ENDIF.

* DataF menor ou igual a datai - pq a data final n pode ser menor que a data inicial
IF dataf <= datai.
  RAISE edata3.
ENDIF.

*Calculo
ndias = dataf - datai.


ENDFUNCTION.
